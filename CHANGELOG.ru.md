# История изменений

> 🇬🇧 [English version](CHANGELOG.md)

## 1.1.0

- Добавлены обязательные ссылки на оригинал для каждой карточки отчёта — карточки без `<a href>` больше не допускаются
- Добавлен Шаг 1.5: разрешение URL через HN Algolia API (`hn.algolia.com/api/v1/search`), WebSearch по заголовку + домену, и конструирование URL из текста ссылок
- Добавлено параллельное разрешение URL (до 8 одновременных WebSearch/WebFetch запросов)
- Добавлен fallback-маркер `🔗 ссылка не найдена` для неразрешённых URL с указанием домена-источника
- Улучшены требования к карточкам: каждая карточка теперь явно требует кликабельную ссылку на оригинальный ресурс

## 1.0.0

- Добавлена навигация по SPA через PinchTab headless-браузер (snapshot'ы accessibility tree)
- Добавлен сбор данных из 3 колонок: Hacker News, GitHub Trending, Lobsters
- Добавлен скрапинг вкладок карусели: Top signals, HN Best Show, GitHub Fastest rising, Popular
- Добавлено параллельное чтение статей через WebFetch (до 8 одновременно)
- Добавлена генерация HTML-отчёта с 7 вкладками: Обзор, GitHub Trending, Hacker News, Lobsters, AI Agents, Deep Reads, Попробовать
- Добавлен переключатель тёмной/светлой темы с сохранением в localStorage
- Добавлены цветные бейджи источников: GitHub (зелёный), HN (оранжевый), Lobsters (красный), AI (фиолетовый)
- Добавлены блоки «Ключевой вывод» для важных статей
- Добавлен responsive CSS grid layout (max-width 1200px)
- Добавлен шаг очистки PinchTab daemon после каждого запуска
- Добавлена структура плагина для marketplace (`claude plugin install`)
